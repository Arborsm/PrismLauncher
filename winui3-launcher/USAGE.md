# Prism Launcher WinUI 3 使用说明

## 快速开始

### 1. 系统要求

- **操作系统**: Windows 10 版本 1903 或更高版本
- **内存**: 至少 4GB RAM
- **存储**: 至少 500MB 可用空间
- **网络**: 用于下载Minecraft版本和模组

### 2. 安装

1. 下载最新版本的安装包
2. 运行安装程序
3. 按照向导完成安装
4. 启动应用程序

### 3. 首次使用

1. **欢迎界面**: 首次启动会显示欢迎界面
2. **创建实例**: 点击"创建新实例"按钮
3. **选择版本**: 选择要安装的Minecraft版本
4. **配置设置**: 设置实例名称和其他选项
5. **开始游戏**: 创建完成后即可启动游戏

## 主要功能

### 实例管理

#### 查看实例列表
- 主界面左侧显示所有实例
- 支持按分组筛选
- 显示实例状态（运行中、离线、已损坏）
- 显示最后游玩时间

#### 创建新实例
1. 点击"添加实例"按钮
2. 选择创建方式：
   - **空白实例**: 创建空的实例
   - **从模板**: 使用预设模板
   - **导入**: 从文件导入
3. 配置实例设置
4. 完成创建

#### 编辑实例
1. 选择要编辑的实例
2. 点击"编辑"按钮
3. 修改设置：
   - 实例名称
   - 图标
   - 分组
   - 游戏设置
4. 保存更改

#### 删除实例
1. 选择要删除的实例
2. 点击"删除"按钮
3. 确认删除操作
4. 实例将被移动到回收站

### 启动游戏

#### 基本启动
1. 选择要启动的实例
2. 点击"启动"按钮
3. 等待游戏加载完成

#### 启动选项
- **在线模式**: 需要登录Microsoft账户
- **离线模式**: 无需登录，使用本地用户名
- **演示模式**: 限制功能的演示版本

#### 服务器连接
1. 在启动前选择"连接到服务器"
2. 输入服务器地址
3. 启动游戏

### 设置和配置

#### 全局设置
1. 点击"设置"按钮
2. 配置选项：
   - **Java设置**: Java路径和参数
   - **网络设置**: 代理和下载设置
   - **界面设置**: 主题和显示选项
   - **账户设置**: Microsoft账户管理

#### 实例设置
1. 选择实例
2. 点击"编辑"按钮
3. 配置实例特定设置

### 模组管理

#### 安装模组
1. 选择实例
2. 进入"模组"标签
3. 点击"添加模组"
4. 选择模组文件或从在线源安装

#### 管理模组
- 启用/禁用模组
- 更新模组版本
- 删除不需要的模组
- 查看模组信息

## 界面说明

### 主窗口布局

#### 标题栏
- **Logo**: Prism Launcher标识
- **搜索框**: 快速搜索实例
- **操作按钮**: 添加实例、设置等

#### 侧边栏
- **分组列表**: 按分组显示实例
- **筛选选项**: 按状态筛选实例
- **实例列表**: 显示所有实例

#### 主内容区
- **欢迎页面**: 未选择实例时显示
- **实例详情**: 选中实例时显示详细信息
- **快速操作**: 启动、编辑、复制等按钮

#### 状态栏
- **状态信息**: 当前操作状态
- **统计信息**: 实例数量和运行状态

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 创建新实例 |
| Ctrl+E | 编辑选中实例 |
| Ctrl+D | 删除选中实例 |
| Ctrl+C | 复制选中实例 |
| F5 | 刷新实例列表 |
| F1 | 显示帮助 |
| Ctrl+, | 打开设置 |

### 右键菜单

在实例上右键点击可访问：
- 启动实例
- 编辑实例
- 复制实例
- 删除实例
- 导出实例
- 查看日志
- 打开文件夹

## 高级功能

### 批量操作

#### 批量选择
1. 按住Ctrl键选择多个实例
2. 右键点击选择批量操作
3. 选择要执行的操作

#### 批量操作类型
- 批量启动
- 批量停止
- 批量删除
- 批量导出

### 导入和导出

#### 导出实例
1. 选择要导出的实例
2. 点击"导出"按钮
3. 选择导出格式：
   - **ZIP文件**: 完整实例备份
   - **MrPack**: 模组包格式
   - **Flame包**: CurseForge格式

#### 导入实例
1. 点击"添加实例"按钮
2. 选择"导入"选项
3. 选择要导入的文件
4. 配置导入设置

### 日志和调试

#### 查看日志
1. 选择实例
2. 点击"查看日志"按钮
3. 查看启动日志和错误信息

#### 调试模式
1. 在设置中启用调试模式
2. 查看详细的调试信息
3. 导出调试报告

## 故障排除

### 常见问题

#### 游戏无法启动
1. 检查Java是否正确安装
2. 验证实例设置
3. 查看错误日志
4. 尝试重新创建实例

#### 模组冲突
1. 检查模组兼容性
2. 更新模组版本
3. 禁用冲突模组
4. 查看模组文档

#### 性能问题
1. 调整Java内存设置
2. 关闭不必要的程序
3. 更新显卡驱动
4. 降低游戏设置
